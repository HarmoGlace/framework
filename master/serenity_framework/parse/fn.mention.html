pub fn mention<'a>(msg: &'a str, id: &str) -> Option<(&'a str, &'a str)>
Expand description

Parses a mention from the message.

A mention is defined as text starting with <@, which may be followed by !, proceeded by a user id, and ended by a >.

This can be expressed in a regular expression as <@!?\d+>.

As an example, these are valid mentions:

  • <@110372470472613888>
  • <@!110372470472613888>

Returns the mention and the rest of the message after the mention, with trimmed whitespace.

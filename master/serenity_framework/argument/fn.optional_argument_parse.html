Expand description

Tries to take a single segment from a list of segments and parse an argument out of it using the serenity::utils::Parse trait.

If the list of segments is empty, Ok(None) is returned. Otherwise, the first segment is taken and parsed into an argument. If parsing succeeds, Ok(Some(...)) is returned, otherwise Err(...). The error is wrapped in ArgumentError::Argument.

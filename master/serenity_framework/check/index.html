Module serenity_framework::check[−][src]

Functions and types relating to checks.

A check is a function that can be plugged into a command to allow/deny a user's access. The check returns a Result that indicates whether it succeeded or failed. In the case of failure, additional information can be given, a reason, that describes the failure.
